# mk8dLivesplit

This code generates split time distribution plots for Mario Kart 8 Deluxe speedruns saved with [LiveSplit](https://one.livesplit.org/).
The code has been tested with 48 tracks runs, although it should work with runs of any length.

For more information, read my [blogpost](https://chipdelmal.github.io/blog/).

This plot generates the distributions of each one of the splits.
